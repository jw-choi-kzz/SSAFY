http://localhost:3000/  
*index.js = 리액트 입구 파일 && App.js는 화면 구성 && App.js 안에 있는 App 함수는 화면 구성 중*   
```index.js``` : 여러 전역 설정 포함  
```App.js``` : UI  
```App.css``` : 페이지 디자인  
```index.css``` : 샘플앱 기본 css  
   
> ```npx serve -s build``` : 서비스 화면
