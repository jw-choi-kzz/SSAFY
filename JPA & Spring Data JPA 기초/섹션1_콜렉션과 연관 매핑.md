# 08 값 콜렉션 Set 매핑
단순 값을 Set으로 보관하는 모델
*집합 형태의 데이터를 갖고 있는 **콜렉션 테이블***  
ex. 원래 role(id, name) 테이블, role_perm(role_id, perm) 테이블 2개 => Role(id, name, **permission: Set<String>**)으로 하나의 테이블로 만들 수 있음  
## 단순 값 Set 매핑
```
@Entity
@Table(name= "role")
public class Role{
  @Id
  private String
}
```
